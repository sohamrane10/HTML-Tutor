HTML Introduction

The backbone of the modern web. Learn how it all started and why it matters.

What is HTML?

HTML stands for HyperText Markup Language. It is the standard markup language for documents designed to be displayed in a web browser. It is NOT a programming language, but a markup language used to structure content.

A Basic HTML Document

Every HTML page starts with a basic structure. Here is how it looks:

html 
<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>

</body>
</html>

Why Learn HTML?

HTML is the starting point for anyone wanting to become a web developer. It provides the structure that CSS (styling) and JavaScript (interactivity) build upon.

  • Universally used across the web.
  • Easy to learn and understand.
  • Essential for SEO and accessibility.
