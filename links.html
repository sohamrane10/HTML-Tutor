HTML Links

Learn how to create links and navigate between pages or websites.

Hyperlinks

HTML links are hyperlinks. You can click on a link and jump to another document or another section of the same document.

html 
<a href="https://www.google.com">This is a link to Google</a>

Link Target Attribute

The target attribute specifies where to open the linked document.

  • _self: The default. Opens the document in the same window/tab as it was clicked.
  • _blank: Opens the document in a new window or tab.
  • _parent: Opens the document in the parent frame.
