HTML Images

Learn how to embed and optimize images on your web pages.

Embedding Images

To embed an image in a webpage, you use the <img> element. It is an empty element that contains attributes only.

html 
<img src="image.jpg" alt="Description of image" width="500" height="300">

Image Attributes

Essential attributes for images include:

  • src: Specifies the path to the image.
  • alt: Provides an alternate text for an image if the image cannot be displayed.
  • width and height: Specifies the width and height of an image.
