HTML Tables

Learn to present structured data using semantic HTML table elements.

Creating a Simple Table

An HTML table is defined by <table>, <tr> (table row), <th> (table header), and <td> (table data/cell).

html 
<table>
    <tr>
        <th>Company</th>
        <th>Contact</th>
        <th>Country</th>
    </tr>
    <tr>
        <td>Alfreds Futterkiste</td>
        <td>Maria Anders</td>
        <td>Germany</td>
    </tr>
    <tr>
        <td>Centro comercial Moctezuma</td>
        <td>Francisco Chang</td>
        <td>Mexico</td>
    </tr>
</table>

Adding Table Headers

Use the <thead>, <tbody>, and <tfoot> elements to group your table’s rows into header, body, and footer content.
