HTML Elements

The building blocks of HTML. Understand tags, attributes, and content.

What are HTML Elements?

An HTML element is defined by a start tag, some content, and an end tag. Elements are what define the structure and layout of a web document.

syntax 
<tagname> Content goes here... </tagname>

Nested HTML Elements

HTML elements can be nested. This means elements can contain other elements. For example, the <body> element contains multiple <h1> and <p> tags.

html 
<div>
    <h1>Main Heading</h1>
    <p>This is a paragraph inside a div.</p>
</div>

Empty HTML Elements

Some elements do not have an end tag. These are called empty elements. Examples include <br> for a line break and <hr> for a horizontal rule.
